Getting Started:
1. Install dependencies: npm install
2. Start development server: npm run dev
3. Start backend server: npm run server
4. Or run both: npm run dev:full

The project is structured for scalability with separate frontend/backend directories and comprehensive documentation for database integration options.

Choose your preferred database solution and follow the respective setup guide:
- SUPABASE_SETUP.md for Supabase
